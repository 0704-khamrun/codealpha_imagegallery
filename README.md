# 🖼️ Image Gallery Frontend Project

A modern and responsive Image Gallery web application built using **HTML, CSS, and JavaScript** (or React, if applicable). This project allows users to browse and view images in an attractive gallery layout with a smooth user experience.

## 🚀 Features

* Responsive image gallery layout
* Clean and modern UI design
* Image preview/lightbox functionality
* Smooth hover effects and animations
* Mobile-friendly design
* Fast and lightweight implementation

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
